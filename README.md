# Pt12-Bimetallic-DFT-Study
This DFT study investigates Li/Na/K-doped Pt12 clusters using Dmol3. Results show K@Pt12 is most stable (Eb=-4.55eV), while Li@Pt12 and Na@Pt12 exhibit small bandgaps (0.13-0.19eV) suitable for photovoltaics. Charge transfer reveals ionic/metallic bonding patterns.
